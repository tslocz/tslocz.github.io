Packages in R

fejiv. R package for fixed effect jackknife IV (FEJIV) estimation (with Qihui Lei). To download from CRAN, type “install.packages("fejiv")” in R/R Studio. See also https://CRAN.R-project.org/package=fejiv.

hettreatreg. R package to compute diagnostics for linear regression when treatment effects are heterogeneous (with Mark McAvoy). To download from CRAN, type “install.packages("hettreatreg")” in R/R Studio. See also https://CRAN.R-project.org/package=hettreatreg.

Packages in Stata

fejiv. Stata package for fixed effect jackknife IV (FEJIV) estimation (with Qihui Lei). To download from SSC, type “ssc install fejiv” in Stata.

hettreatreg. Stata package to compute diagnostics for linear regression when treatment effects are heterogeneous. To download from SSC, type “ssc install hettreatreg, all” in Stata.

kappalate. Stata package to estimate the local average treatment effect (LATE) using Abadie's kappa approach and other weighting estimators (with S. Derya Uysal and Jeffrey M. Wooldridge). To download from SSC, type “ssc install kappalate” in Stata.

teffects2. Stata package to estimate average treatment effects with observational data (with S. Derya Uysal and Jeffrey M. Wooldridge). To download from SSC, type “ssc install teffects2, all” in Stata.
